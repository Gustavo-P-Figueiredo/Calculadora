# Calculadora

## Proposta
Desenvolver um projeto simples de operações em uma calculadora, nas seguintes condições: O desenvolvimento de cada Commit será feito em até 20 minutos, e se nescessario pesquisa externa para tirar duvidas no desenvolvimento será feito em até 10 minutos :clock1:

### Objetivos
Permitir as quatro operações basicas (Adição, subtração, multiplicação e divisão) :white_check_mark:
Armazenar os ultimos cinco resultados :white_check_mark:
Permitir operações avançadas :soon:
Permitir expressões matematicas :soon:
Permitir conversão de valores :soon:
